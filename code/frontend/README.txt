# FloodGuard Frontend

## Overview
This directory contains the frontend web application for the FloodGuard flood early-warning system.
Provides real-time visualization of water levels, flood risk predictions, and public advisories.

## Files
- **index.html**: Main HTML structure for the dashboard
- **app.js**: Core application logic and chart rendering
- **style.css**: Responsive design and UI styling
- **package.json**: Project dependencies and build scripts

## Technologies
- HTML5 / CSS3
- Vanilla JavaScript
- Chart.js for data visualization
- Material Icons for UI components

## Running the Application
```bash
npm install
npm run dev    # Development server
npm run build  # Production build
```
